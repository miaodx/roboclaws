## Notes

- `mimo-inside-openai-chat:mimo-1000` is the clear throughput winner in this
  run, but it remains an on-demand benchmark/text route, not a product cleanup
  default.
- Among current product-relevant routes, `mimo-mify-responses` and
  `mimo-token-plan` were materially faster than Kimi, Codex, MiniMax, and
  Nvidia on throughput.
- Kimi was mid-pack in this run.
- Codex and MiniMax were close on throughput; MiniMax had better first-token
  latency.
- Kimi suffix probes on 2026-06-22 showed arbitrary K2.7 suffixes can return
  HTTP 200 and echo the supplied model name, so `kimi-k2.7-code` is the only
  catalog model id tracked here.
