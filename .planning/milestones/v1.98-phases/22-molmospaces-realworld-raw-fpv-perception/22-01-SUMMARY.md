# 22-01 Real-World Raw FPV Perception Summary

Completed: 2026-05-09

## Outcome

Phase 22 added ADR-0013 raw FPV-only perception evidence mode to the ADR-0003
MolmoSpaces cleanup path.

The default `visible_object_detections` mode remains unchanged. The new
`raw_fpv_only` mode records waypoint-level camera observations without
structured movable-object detections, observed handles, categories, support
estimates, target labels, or generated mess truth. When robot views are
recorded, each raw observation links to the FPV image generated by the existing
RBY1M robot-view underlay.

## Changes

- Added `perception_mode` to the deterministic ADR-0003 harness, MCP smoke, and
  direct agent server.
- Added `raw_fpv_only` branching in `RealWorldCleanupContract.observe()`.
- Added `agent_view.raw_fpv_observations` and top-level
  `run_result["raw_fpv_observations"]`.
- Added MCP/demo artifact attachment from existing `record_robot_view_step`
  captures.
- Added `Raw FPV Observations` to the shared Cleanup Artifact Report.
- Added `--require-raw-fpv-observations` checker support.
- Added `just harness::molmo-realworld-raw-fpv` and
  `just verify::molmo-realworld-raw-fpv`.
- Added contract, MCP, report, checker, demo, and just recipe tests.
