<objective>
Build the in-process FastMCP server that replaces the HTTP sim_server. Exposes observe/move/done as first-class MCP tools over streamable-http on the host (port configurable, default 18788, path `/mcp`). This is the single new module that carries the three-tool contract locked by Phase 2.6 CONTEXT.md D-01.

Purpose: Deliver the tool surface that the Gateway's `profile: minimal` agent will see as `roboclaws__observe` / `roboclaws__move` / `roboclaws__done`. Image content flows through the MCP SDK's `Image` type so the sanitizeToolResultImages → anthropic-messages adapter chain preserves multimodal input to Kimi (spike U1, verified 2026-04-21).

Output: New module + unit tests + dependency declared in pyproject so fresh `pip install -e ".[dev]"` works for a contributor who did not run the spike's `uv pip install "mcp[cli]"` + a trace_schema_reference.json fixture that pins the schema renderers depend on.
