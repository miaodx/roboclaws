    ```markdown
    ## MCP config in openclaw.json

    Gateway's MCP loader (image 2026.4.14, `/app/dist/pi-bundle-mcp-tools-CxZ16DeR.js:128-170`)
    expects this exact shape under `mcp.servers.<name>`:

    ```jsonc
    {
      "transport": "streamable-http",  // literal "transport", not "type"
      "url": "http://host.docker.internal:18788/mcp"
    }
    ```

    Accepted `transport` values: `"streamable-http"` or `"sse"`. Anything else
    (e.g. `"http"`) fails silently — the Gateway logs
    `[bundle-mcp] failed to start server "<name>" (<url>): Error: SSE error: Non-200 status code (400)`
    and the agent simply doesn't see the tools. The `openclaw mcp set` CLI does
    NOT validate; it writes whatever JSON you give it.

    ### Restart gotcha: seed MCP config before first start

    Changing `mcp.servers` on a running Gateway triggers:
    ```
    [reload] config change requires gateway restart (mcp)
    [gateway] signal SIGUSR1 received
    [gateway] restart mode: full process restart (spawned pid 174)
    ```
    In Docker, that PID-1 restart is a container exit (`Exited (0)`). Recovery
    requires `docker start openclaw-gateway`.

    Mitigation: `scripts/openclaw-bootstrap.sh` seeds `mcp.servers` (and the
    matching `agents.list[<n>].tools.profile`) into `openclaw.json` BEFORE the
    first `docker run`. Don't use `openclaw mcp set` on a live container unless
    you mean to restart.

    ### Tool-policy profiles

    `agents.list[<n>].tools.profile` accepts `"minimal"`, `"coding"`, or
    `"messaging"` (source: `/app/dist/tool-policy-CwZyJfFc.js:352-354`).

    - `minimal` — `session_status` + `update_plan` only, plus whatever MCP
      servers expose. No `exec`, no fs, no `image`, no `browser`, no
      `web_fetch`. Phase 2.6 uses this.
    - `coding` — the default. 25 tools including `exec`, `read`, `write`,
      `image`, `browser`. Right for general agent work; wrong for autonomous
      AI2-THOR navigation (Phase 2.5 proved the agent drifts to `exec`).
    - `messaging` — a middle ground. Not used in roboclaws.
