# Plan 02.5-07: scripts/render_autonomous_replay.py — trace → replay.gif + report.html

**Source task in reviewed plan:** T54 (see `docs/retrospectives/plans/phase-2.5-autonomous-nav.md`)
**Owner:** cloud
**Wall estimate:** 2-3 hrs

## Objective

Turn the `trace.jsonl` produced by Plan 02.5-02's SimHTTPServer into
visible artifacts. Two outputs: (1) `replay.gif` — a smooth per-frame
GIF that draws BOTH agent-observed and server-side frames so the replay
looks continuous, regardless of whether the agent batched moves; (2)
`report.html` — a single-file HTML decision audit that shows FPV +
overhead per step, whether the frame was seen by the agent, and whether
continued moves were reasoned or blind.

Create `scripts/render_autonomous_replay.py` with shebang

1. **CLI**:
   ```python
   import argparse, pathlib
   parser = argparse.ArgumentParser()
   parser.add_argument("--run-dir", type=pathlib.Path, required=True,
                       help="Directory containing trace.jsonl")
   parser.add_argument("--thumbnail-size", type=int, default=160,
                       help="Max width of each frame thumbnail in report.html")
   args = parser.parse_args()
   trace_path = args.run_dir / "trace.jsonl"
   assert trace_path.is_file(), f"trace.jsonl not found at {trace_path}"
   ```

2. **Parse trace.jsonl** into a list of events:
   ```python
   import json
   events = []
   for line in trace_path.read_text().splitlines():
       line = line.strip()
       if not line:
           continue
       events.append(json.loads(line))
   ```

3. **Extract frame_capture events in order**:
   ```python
   frames = [ev for ev in events if ev.get("event") == "frame_capture"]
   ```

4. **Decode + composite each frame**. For each frame event, base64-decode
   `fpv` and `overhead`, composite side-by-side using the same approach
   as `roboclaws/core/visualizer.py` (or reuse its helper directly if
   available). Draw the two images side-by-side with a thin border.
   Collect PIL Image objects in order.

5. **Write replay.gif**:
   ```python
   import imageio.v2 as imageio
   import numpy as np
   gif_path = args.run_dir / "replay.gif"
   imageio.mimsave(gif_path,
                   [np.array(img) for img in composite_images],
                   duration=0.15)  # ~6-7 fps — smooth enough to watch
   ```
   BOTH seen_by_agent=true and seen_by_agent=false frames are drawn —
   the GIF shows smooth coverage regardless of observation cadence.

6. **Write summary.json**:
   ```python
   observe_events = [ev for ev in events
                     if ev.get("tool") == "observe" and ev.get("event") == "request"]
   move_events = [ev for ev in events
                  if ev.get("tool") == "move" and ev.get("event") == "request"]
   done_events = [ev for ev in events
                  if ev.get("tool") == "done" and ev.get("event") == "request"]
   seen_frames = [f for f in frames if f.get("seen_by_agent") is True]
   unseen_frames = [f for f in frames if f.get("seen_by_agent") is False]
   human_delivered = sum(1 for ev in events
                         if ev.get("event") == "response"
                         and ev.get("response", {}).get("human_message"))

   total_tool_calls = len(observe_events) + len(move_events) + len(done_events)
   observe_to_move = (
       len(observe_events) / len(move_events)
       if move_events else float("inf")
   )
   wallclock = (frames[-1]["wallclock_elapsed"]
                - frames[0]["wallclock_elapsed"]) if frames else 0.0
   terminated_by = ("done" if done_events else "wall_clock")

   summary = {
       "total_tool_calls": total_tool_calls,
       "tool_calls_by_type": {
           "observe": len(observe_events),
           "move": len(move_events),
           "done": len(done_events),
       },
       "moves": len(move_events),
       "observes_by_agent": len(seen_frames),
       "frames_unseen_by_agent": len(unseen_frames),
       "observe_to_move_ratio": observe_to_move,
       "wallclock_seconds": wallclock,
       "terminated_by": terminated_by,
       "human_messages_delivered": human_delivered,
   }
   (args.run_dir / "summary.json").write_text(json.dumps(summary, indent=2))
   ```

7. **Defer HTML rendering to task 02** (separate task for clarity —
   Jinja2 template + per-frame tags are enough work to isolate).

2. **Wire the renderer** (`scripts/render_autonomous_replay.py`) to
   produce report.html:

   a. Add Jinja2 import (already a transitive dep via other tooling; if
      not present, add `jinja2` to `pyproject.toml` extras for this
      script — confirm before adding).

   b. Build the `timeline` list: walk `frames` in order, insert a
      `{"kind": "batched_warning", "count": N}` entry when N ≥ 5
      consecutive `seen_by_agent=false` frames occur (N is the length
      of the consecutive unseen run).
      ```python
      timeline = []
      unseen_run = 0
      for f in frames:
          if not f.get("seen_by_agent"):
              unseen_run += 1
          else:
              if unseen_run >= 5:
                  timeline.append({"kind": "batched_warning",
                                   "count": unseen_run})
              unseen_run = 0
          timeline.append({
              "kind": "frame",
              "seen_by_agent": bool(f.get("seen_by_agent")),
              "ts": f.get("wallclock_elapsed", 0.0),
              "fpv": f.get("fpv", ""),
              "agent_state": f.get("agent_state", {}),
              "human_message": f.get("human_message"),
          })
      if unseen_run >= 5:
          timeline.append({"kind": "batched_warning", "count": unseen_run})
      ```

   c. Build `tool_log` list: walk `events`, filter to tool requests and
      responses, summarize each line:
      ```python
      tool_log = []
      for ev in events:
          if ev.get("event") == "request" and ev.get("tool"):
              tool = ev["tool"]
              summary = ""
              if tool == "move":
                  summary = f"direction={ev.get('request', {}).get('direction', '?')}"
              elif tool == "done":
                  summary = f"reason={ev.get('request', {}).get('reason', '?')}"
              tool_log.append({
                  "ts": ev.get("wallclock_elapsed", 0.0),
                  "tool": tool,
                  "summary": summary,
                  "human_message": None,
              })
          elif ev.get("event") == "response":
              msg = ev.get("response", {}).get("human_message")
              if msg and tool_log:
                  tool_log[-1]["human_message"] = msg
      ```

   d. Render the template:
      ```python
      from jinja2 import Environment, FileSystemLoader
      template_dir = pathlib.Path(__file__).parent / "templates"
      env = Environment(loader=FileSystemLoader(str(template_dir)),
                        autoescape=True)
      template = env.get_template("autonomous_report.html.j2")
      html = template.render(
          run_id=args.run_dir.name,
          summary=summary,
          timeline=timeline,
          tool_log=tool_log,
          thumbnail_size=args.thumbnail_size,
      )
      (args.run_dir / "report.html").write_text(html, encoding="utf-8")
      ```
