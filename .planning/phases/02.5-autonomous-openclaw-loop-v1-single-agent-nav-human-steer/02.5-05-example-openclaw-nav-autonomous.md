# Plan 02.5-05: examples/openclaw_nav_autonomous.py — end-to-end autonomous nav demo

**Source task in reviewed plan:** T53 (see `docs/plans/phase-2.5-autonomous-nav.md`)
**Owner:** cloud (code); local-dev exercises it end-to-end in Plan 02.5-08
**Wall estimate:** 2-3 hrs

## Objective

The user-facing demo script that stitches together everything from
Plans 02.5-02 through 02.5-07: boots AI2-THOR, starts the sim HTTP
server, bootstraps the Gateway, wires up stdin-based human interjection,
fires one blocking `start_run` kickoff, renders artifacts on return, and
tears down cleanly on SIGINT. This is the script the local-dev
validation probes in Plan 02.5-08 execute.

<action>
Create `examples/openclaw_nav_autonomous.py` with `#!/usr/bin/env python3`
and executable permissions. Structure:

1. **CLI + argparse**:
   ```python
   import argparse, datetime as dt, pathlib
   parser = argparse.ArgumentParser()
   parser.add_argument("--scene", default="FloorPlan201")
   parser.add_argument("--max-moves", type=int, default=200)
   parser.add_argument("--wall-budget", type=float, default=600.0)
   parser.add_argument("--output-dir", type=pathlib.Path, default=None)
   args = parser.parse_args()
   run_id = args.output_dir or pathlib.Path(
       f"output/openclaw-autonomous/{dt.datetime.utcnow().strftime('%Y%m%dT%H%M%SZ')}"
   )
   run_id.mkdir(parents=True, exist_ok=True)
   ```

2. **Resource allocation + try/finally skeleton**:
   ```python
   import threading, logging, signal, sys, subprocess
   logging.basicConfig(level=logging.INFO,
                       format="[%(asctime)s] %(levelname)s %(message)s")
   log = logging.getLogger("openclaw-nav-autonomous")

   engine = None
   sim_server = None
   bridge = None
   stdin_thread = None
   stdin_stop = threading.Event()
   gateway_started = False
   try:
       # a. Start AI2-THOR engine
       from roboclaws.core.engine import MultiAgentEngine
       log.info("starting MultiAgentEngine(scene=%s, agent_count=1)", args.scene)
       engine = MultiAgentEngine(scene=args.scene, agent_count=1)

       # b. Start sim HTTP server
       from roboclaws.openclaw.sim_server import SimHTTPServer
       sim_server = SimHTTPServer(engine, agent_id=0, run_dir=run_id, port=18788)
       log.info("SimHTTPServer listening on http://127.0.0.1:18788")

       # c. Bootstrap Gateway
       env = {**__import__("os").environ,
              "TIMEOUT_SECONDS": str(int(args.wall_budget + 60)),
              "SIM_SERVER_URL": "http://host.docker.internal:18788"}
       log.info("bootstrapping Gateway (TIMEOUT_SECONDS=%d)",
                int(args.wall_budget + 60))
       result = subprocess.run(
           ["./scripts/openclaw-bootstrap.sh"],
           capture_output=True, text=True, env=env, check=True)
       token = result.stdout.strip()
       gateway_started = True
       log.info("Gateway started, bearer token captured")

       # d. stdin reader thread if TTY
       if sys.stdin.isatty():
           def _stdin_pump():
               while not stdin_stop.is_set():
                   line = sys.stdin.readline()
                   if not line:
                       break
                   line = line.strip()
                   if line:
                       sim_server.enqueue_human_message(line)
                       log.info("queued human message: %s", line[:80])
           stdin_thread = threading.Thread(
               target=_stdin_pump, daemon=True, name="stdin-pump")
           stdin_thread.start()
           log.info("stdin reader thread started (type a line then Enter to interject)")
       else:
           log.info("interjection disabled (no TTY)")

       # e. Build bridge and call start_run
       import os as _os
       from roboclaws.openclaw.bridge import OpenClawBridge
       bridge = OpenClawBridge(
           gateway_url="http://127.0.0.1:18789",
           token=token,
       )
       kickoff = (
           f"You are navigating a simulated indoor room. "
           f"You have up to {args.max_moves} physical moves before the run ends.\n\n"
           f"Use the observe, move, done tools declared in your skill. "
           f"Watch for human_message in tool responses (see skill docs for contract).\n\n"
           f"Start by calling observe. Explore the room. Report what you see as you go."
       )
       log.info("calling OpenClawBridge.start_run (wall_budget=%.0fs)", args.wall_budget)
       run_result = bridge.start_run(
           agent_id=0,
           prompt=kickoff,
           wall_budget_s=args.wall_budget,
           done_event=sim_server.done_event,
       )
       log.info("start_run returned: terminated_by=%s wallclock=%.1fs",
                run_result.terminated_by, run_result.wallclock_s)

       # f. Render artifacts
       log.info("rendering replay.gif + report.html")
       subprocess.run(
           ["python", "scripts/render_autonomous_replay.py",
            "--run-dir", str(run_id)],
           check=False,
       )
       log.info("artifacts at %s", run_id)

   finally:
       # g. Teardown in reverse order
       log.info("teardown: begin")
       if stdin_thread is not None:
           stdin_stop.set()
           # daemon thread will exit on process exit; don't block shutdown
           log.info("teardown: stdin thread stop signal set")
       if bridge is not None:
           try:
               bridge.close()
           except Exception as exc:
               log.warning("teardown: bridge.close() failed: %s", exc)
           log.info("teardown: bridge closed")
       if sim_server is not None:
           try:
               sim_server.close()
           except Exception as exc:
               log.warning("teardown: sim_server.close() failed: %s", exc)
           log.info("teardown: sim_server closed")
       if gateway_started:
           try:
               subprocess.run(
                   ["docker", "rm", "-f", "openclaw-gateway"],
                   capture_output=True, check=False)
           except Exception as exc:
               log.warning("teardown: docker rm failed: %s", exc)
           log.info("teardown: Gateway container removed")
       if engine is not None:
           try:
               engine.close()
           except Exception as exc:
               log.warning("teardown: engine.close() failed: %s", exc)
           log.info("teardown: engine closed")
       log.info("teardown: done")
   ```

3. **Write a summary.json** after rendering (outside the finally — only
   on successful completion):
   ```python
   # Inside the try block, after render:
   import json
   summary = {
       "scene": args.scene,
       "max_moves": args.max_moves,
       "wall_budget_s": args.wall_budget,
       "terminated_by": run_result.terminated_by,
       "wallclock_s": run_result.wallclock_s,
       "final_message": run_result.final_message,
       "run_dir": str(run_id),
   }
   (run_id / "summary.json").write_text(json.dumps(summary, indent=2))
   ```
   (The richer summary — observes_by_agent, tool-call breakdown etc. —
   is written by scripts/render_autonomous_replay.py from trace.jsonl.
   This script writes the one-line run-level summary.)

4. **SIGINT handling**: because the entire body is inside `try/finally`,
   Ctrl+C raises KeyboardInterrupt which propagates through the try and
   triggers the finally block — no explicit signal handler needed.
   Confirm logs show teardown order on Ctrl+C.

5. **Line length 100, type annotations on helpers, `from __future__
   import annotations` at the top.** Shebang `#!/usr/bin/env python3`.
</action>

## Rescue Paths

| Failure | Rescue |
|---|---|
| User hits Ctrl+C mid-run | try/finally tears down sim server → Gateway container → daemon threads in order. Plan 02.5-08 step 6 tests this end-to-end. |
| stdin thread blocks on a pipe (no TTY) | `sys.stdin.isatty() == False` on startup → reader doesn't start → log "interjection disabled (no TTY)" and continue. |
| Gateway bootstrap exits non-zero | `subprocess.run(..., check=True)` raises CalledProcessError → try block exits → finally runs → engine + sim_server cleaned up. Error surfaced to user via traceback. |
| Kickoff call exceeds wall_budget_s + 60 | start_run returns terminated_by="wall_clock" with partial final_message. Artifacts still render from trace.jsonl. |
| Agent calls move before first observe | Non-fatal per Plan 02.5-02: server_warning appears in trace + response. Agent proceeds. |
| Memory accumulation across back-to-back runs | Plan 02.5-04's start_run wipes workspaces/agent-0/state/* before POST. Plan 02.5-08 step 5 is the regression probe. |
