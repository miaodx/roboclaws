## Result

Completed on 2026-05-10.

The local run selected four generated fallback requests and executed all four
with RBY1M/CuRobo warmup. Warmup got through config import, and every proof
attempt reached task sampling instead of timing out at `rby1m_config_import`.

  `HouseInvalidForTask`.
- `proof_001_fallback_02` and `proof_002_fallback_02` blocked with
  `AssertionError: Object is not a root body`.

The proof result summary reported `planner_backed_count=0`,
`cleanup_binding_promoted_count=0`, `timeout_count=0`, and
`view_artifact_count=0`. Planner-backed cleanup readiness remains blocked.
