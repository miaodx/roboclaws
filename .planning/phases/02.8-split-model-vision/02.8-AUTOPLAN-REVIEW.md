# Phase 02.8 Autoplan Review

## Verdict

Approved with auto-fixes.

The phase is worth doing. The repo already advertises text-only MiMo entry
points, but the live `roboclaws__observe` contract is still raw-image-first
and `profile: minimal` does not expose the generic Gateway `image` tool. That
is a real operator-contract bug, not a vanity refactor.

The issues were in proof strength and contract specificity, not in the core
direction. The review tightened the spike so it can actually reject or accept
the Gateway `image` path, pinned the exact text-bridge observe shape, fixed the
validation/test-map drift, and made the local-dev gate prove value as well as
mere existence.

## Findings Fixed During Review

1. `02.8-01-profile-probe-and-bridge-contract-spike-PLAN.md` could have chosen
   `gateway-image-tool-primary` without ever proving the agent-side `image`
   path worked for the real two-image navigation case.
   The fix adds a required live usability probe whenever `image` is exposed.

2. `02.8-02-mcp-observe-text-bridge-PLAN.md` under-specified the agent-facing
   observe contract.
   The fix pins the delivery shape to exactly two text blocks in bridge mode,
   names `ROBOCLAWS_OBSERVE_MODE`, and requires truthful
   `image_labels=["vision_bridge"]`.

3. `02.8-VALIDATION.md` and `02.8-04-local-dev-validation-and-doc-update-PLAN.md`
   drifted from the planned file/test surface.
   The fix corrects the spike helper path, adds
   `tests/test_openclaw_vision_bridge.py` to the focused slice, and fixes
   threat references.

4. `02.8-04-local-dev-validation-and-doc-update-PLAN.md` could have passed on
   one lucky move and could only repair `mcp_server.py` even if the failure
   lived elsewhere in the shipped split-model path.
   The fix raises the local gate to a non-trivial run, records bridge latency,
   and widens remediation to the actual blast radius.

5. The outside Codex voice made one valid strategic push: prove the split path
   against a direct-vision control, not just against nothing.
   I accepted the cheap part of that argument without splitting the phase: the
   local-dev gate now includes one `mimo-v2-omni` control run on the same
   scene/budget.
